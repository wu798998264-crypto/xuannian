# 玄念

本地剪切板、收藏、置顶便签与全盘文件查找工具。收藏和附件继续使用稳定的数据目录，覆盖安装新版本不会删除已有数据。

收藏页面支持右键分类名称直接修改或删除；删除前会确认，分类内收藏自动移动到剩余分类，至少保留一个分类。

## 下载

请在 [最新版本发布页](https://github.com/wu798998264-crypto/xuannian/releases/latest) 按设备选择：

- Windows 安装版：`XuanNian-版本号-Setup.exe`
- Windows 便携版：`XuanNian-版本号-Portable.exe`
- Apple silicon Mac（M1/M2/M3/M4）：`XuanNian-版本号-arm64.dmg`
- Intel Mac：`XuanNian-版本号-x64.dmg`

不要下载 `latest.yml`、`latest-mac.yml`、`blockmap` 文件；它们供自动更新使用。

## 全盘查找

第三个主板块可按名称或路径查找文件与文件夹，并支持文件、文件夹、文档、图片、视频、音频筛选，以及排序、打开、定位和复制路径。文档筛选覆盖常见文本、PDF、Office/WPS、OpenDocument、iWork 和电子书格式。图片和视频结果会在搜索完成后按可见区域异步加载系统缩略图，悬浮缩略图可放大预览，鼠标离开后自动关闭。默认快捷键为 `Ctrl+Alt+A`，可在设置中修改；截图默认快捷键相应调整为 `Ctrl+Alt+D`，原有自定义截图快捷键不会被覆盖。

缩略图不参与文件查询：结果先显示，只为当前视窗及其后 10 条结果异步生成预览。滚动后以最终停留视窗为最高优先级，快速跳过的区间会从队列移除，随后完成的过期结果也不会写入缓存。预览并发限制为 3，后台未完成请求限制为 9，缓存上限为 192 项。可视范围内的缩略图若遇到系统服务临时失败，会按退避间隔自动重试，不需要再次滚动；失败结果不会永久写入缓存。系统缩略图和玄念内部解码器都无法生成画面时，文件名前会明确显示“不可预览”；后台仍会低频重试，后来成功时自动移除提示并显示缩略图。双击搜索结果横条可直接打开对应文件或文件夹。

Windows 首次启用会由“玄念全盘查找初始化”请求一次系统授权，用于安装本机文件索引服务；权限窗口不会再显示为 Everything 发起。索引和查询均在本机完成，不读取或上传文件内容。macOS 直接使用系统 Spotlight 索引，无需额外安装服务。

Windows 版内含按 MIT License 再分发的 Everything 与 ES，许可证随安装资源一并提供。

## 运行性能

快捷窗口隐藏时只记录数据已变化，不再后台重绘；再次唤出时立即合并刷新。剪切板使用统一事件代理和渲染快照，相同数据不会重复创建整页节点；离开顶部剪切板页面后会释放多余节点和媒体资源。文件图标只在接近可见区域时加载，并发上限为 4、缓存上限为 256 项。

耐久回归会使用 500 条剪切板、10000 条收藏、10000 条文件结果和 300 个系统图标请求，检查渲染时间、DOM 上限、任务并发、缓存上限以及垃圾回收后的内存是否继续增长。媒体库另提供默认两小时的连续切换、搜索、滚动与下载进度耐久测试；本地可运行 `npm run test:endurance`，短时回归使用 `npm run test:endurance:smoke`。

媒体列表采用可视区域虚拟渲染，大量下载和收藏不会一次性创建全部文件行。媒体库左侧将“已下载/收藏”合并为独立按钮组并与“视频/音乐”保持间隔；选中已下载或收藏时，视频和音乐不再同时高亮。右侧始终显示下载记录入口；视频、歌曲宝普通音质音乐及夸克/阿里云盘高清音乐都使用同一下载记录列表。普通音乐显示准备、传输和完成状态；云盘客户端无法提供真实字节进度时显示等待状态，检测到本地文件后自动完成。已完成任务最多保留最近 10 条，可一键清空历史记录且不会删除磁盘文件或中断现有任务，完整文件仍可在“已下载”中查看。记录单独存放，不写入或修改收藏主数据。嵌入式下载网站离开三分钟后释放网页进程，导航历史最多保留 20 条，HTTP 缓存超过 128 MB 时自动清理；这些操作不删除媒体下载或媒体收藏文件。

小红书媒体地址使用其 CDN 要求的来源信息下载，Seekin 后台解析完成后可继续生成临时预览并执行正式下载。下载记录显示已知文件大小，“等待重试”任务可直接删除；视频收藏夹选择器底部可新建分类。视频下载按钮会在原生网页层隐藏后接管完整可见区域，旧页面的新手提示不会在媒体页透明遮挡点击。

主窗口、快捷窗口、置顶便签和弹窗都保留浏览器原生滚轮行为；仅当某次滚轮输入没有带动可滚动内容时才执行一次兜底滚动，不会重复位移或创建常驻定时器。全盘查找的图片与视频缩略图通过统一 API 异步请求，结果列表无需等待缩略图即可显示，也不会为视窗外的完整结果集预先生成缩略图。

## 版本规则

发布版本统一使用 `主版本.次版本.修订号`。次版本和修订号只能为 `0` 到 `9`：`x.y.9` 的下一版进位为 `x.(y+1).0`，`x.9.9` 的下一版进位为 `(x+1).0.0`，不再使用两位数的末级版本号。
